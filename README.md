Getting Started
Prerequisites

    Node.js (v18+)
    npm or Bun

Installation

# Clone the repository
git clone https://github.com/saidul-122/wellspring-web-application/

# Navigate to project directory
cd wellspring

# Install dependencies
npm install

# Start development server
npm run dev
